# 📊 FedEx Logistics Data Analysis

## 🎯 Project Overview
This project analyzes the SCMS Delivery History dataset to identify shipment delays, optimize logistics operations, and reduce costs.

---

## 🧠 Business Objective
Improve delivery performance, reduce freight cost, and enhance supply chain efficiency using data-driven insights.

---

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly

---

## 📊 Key Analysis Performed
- Data Cleaning & Preprocessing
- Feature Engineering (Delivery Delay, Shipment Time)
- Exploratory Data Analysis (EDA)
- Visualization (10+ charts)
- Correlation Analysis

---

## 🔍 Key Insights
- Air shipments have highest on-time delivery rate
- Shipment mode significantly impacts cost and delay
- Weight and shipment time influence freight cost
- Vendor performance affects delivery efficiency
- Certain countries show higher delays

---

## 📈 Recommendations
- Optimize shipment mode selection
- Prioritize high-performing vendors
- Improve operations in high-delay regions
- Use data-driven monitoring

---

## 🎯 Conclusion
This project demonstrates how data analytics can improve logistics efficiency, reduce costs, and support better decision-making.
